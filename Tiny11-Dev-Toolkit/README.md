## 使用说明

### RegistryTweaks 文件夹

每个子文件夹包含一对注册表文件：
- **Enable.reg / Add.reg** - 应用该调整
- **Disable.reg / Remove.reg** - 撤销该调整
- **README.md** - 详细说明

### 如何使用 .reg 文件

1. 双击 .reg 文件
2. 在弹出的确认对话框中点击"是"
3. 重新启动资源管理器或重启电脑使更改生效

### 重要提示

- 修改注册表前建议先创建系统还原点
- 某些更改需要注销或重启才能生效
- 如遇问题，运行对应的撤销 .reg 文件即可恢复
