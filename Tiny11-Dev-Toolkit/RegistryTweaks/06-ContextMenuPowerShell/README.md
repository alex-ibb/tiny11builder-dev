# PowerShell here 右键菜单

## 功能说明

在文件夹右键菜单中添加 "PowerShell here" 选项，可以快速在当前目录打开 PowerShell。

## 默认状态

**已添加**

## 文件说明

- `Add.reg` - 添加 "PowerShell here" 菜单
- `Remove.reg` - 移除 "PowerShell here" 菜单

## 使用方法

- 在文件夹空白处右键 → "PowerShell here"
- 在文件夹上右键 → "PowerShell here"

## 支持的路径

- 普通路径、带空格路径、中文路径均可正常工作
- 注意：路径中不能包含单引号字符

## 生效方式

应用后**立即生效**。
