# Edge 浏览器优化

## 功能说明

通过组策略禁用 Edge 浏览器的一些"干扰"功能。

## 默认状态

**已优化**

## 禁用的功能

| 功能 | 效果 |
|------|------|
| 新标签页新闻 | 无推送内容 |
| 快速链接推荐 | 无网站推荐 |
| 侧边栏 (Copilot) | 无 AI 弹窗 |
| 首次运行体验 | 跳过引导流程 |
| 购物助手 | 无购物提示 |
| 集锦功能 | 简化界面 |
| 关注功能 | 无社交推荐 |

## 文件说明

- `Optimize.reg` - 应用优化设置 (需管理员权限)
- `Default.reg` - 删除策略，恢复默认 (需管理员权限)

## 生效方式

应用后**重启 Edge**生效。
