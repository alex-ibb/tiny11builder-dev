# PowerShell (Admin) 右键菜单

## 功能说明

在文件夹右键菜单中添加 "PowerShell here (Admin)" 选项，可以快速以管理员权限在当前目录打开 PowerShell。

## 默认状态

**已添加**

## 文件说明

- `Add.reg` - 添加 "PowerShell here (Admin)" 菜单
- `Remove.reg` - 移除 "PowerShell here (Admin)" 菜单

## 使用方法

- 在文件夹空白处右键 → "PowerShell here (Admin)"
- 系统会弹出 UAC 提示，点击"是"后打开管理员 PowerShell

## 注意事项

- 菜单项旁会显示 UAC 盾牌图标
- 需要管理员权限才能正常运行

## 生效方式

应用后**立即生效**。
